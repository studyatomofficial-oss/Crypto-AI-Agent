# Crypto AI Agent

A simple crypto analysis agent scaffold for collecting market data, computing indicators, applying filters, and generating reports.

## Structure

- API integrations live in the `api/` package.
- Market and scoring logic lives in `services/`.
- Technical indicators are implemented in `indicators/`.
- Reporting tools are in `reports/`.

## Getting started

1. Create and activate a virtual environment.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the entry point: `python main.py`
